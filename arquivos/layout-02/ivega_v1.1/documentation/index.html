Files Structure

- bootstrap  (All Bootstrap files. We keep all of them in this folder to make updates easily)
- css  (CSS files of the template)
- - skins  (css skin files)
- fonts (external font libraries)
- - font-awesome
- images  (all the images of the template)
- js  (Javascript files of the template)
- plugins  (All external libs. We keep all of them in this folder to make updates easily.)

Html Structure

The template is based on Bootstrap 3 Framework - http://getbootstrap.com

Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases.

If you need more information, please visit this site: http://getbootstrap.com/css/#grid

CSS Structure

Folder bootstrap

This folder contains all the Bootstrap 3 stylesheets

- bootstrap/css/bootstrap.css (Generic styles, responsive grid styles, resets & normalization)
 

Folder css

This folder contains the stylesheets of the template
 
- css/style.css (This stylesheet contains all the style rules of the template)
- css/animations.css (This stylesheet contains all the css rules for the animations of the template)
- css/custom.css (In this stylesheet you should put your own custom css rules)
 

Folder css/skins

This folder contains all the color schemes of the template

- css/skins/blue.css (blue skin)
- css/skins/red.css (red skin)
- css/skins/green.css (green skin)
- css/skins/pink.css (pink skin)
- css/skins/gray.css (gray skin)
- css/skins/purple.css (purple skin)
- css/skins/orange.css (orange skin)
- css/skins/cyan.css (cyan skin)

Javascript

The initialization of the elements, libs and features is made by the file template.js, in the JS folder.

All external js libraries are in the plugins folder. We keep all of them in this folder to make updates easily.

Customization & Styles

Skins

To change the default color scheme, please replace the blue.css to the following tag in your head:

<link href="css/skins/blue.css" rel="stylesheet">

Layout Style

The template has two different layout modes, Wide and Boxed. The wide one is the default option, if you want to use the Boxed version, add the boxed class in the page-wrapper <div> as below:

<div class="page-wrapper boxed">

Fonts

To change the embedded fonts, please take a look in the head part of the website and you will find these tags:

<link href="http://fonts.googleapis.com/css?family=Lato:300,400,700,900,300italic,400italic,700italic" rel="stylesheet" type="text/css">
<link href="http://fonts.googleapis.com/css?family=Open+Sans:400italic,700italic,400,700,300&amp;subset=latin,latin-ext" rel="stylesheet" type="text/css">
<link href="http://fonts.googleapis.com/css?family=PT+Serif" rel="stylesheet" type="text/css">

To change the fonts, visit the http://www.google.com/webfonts choose a font and use the generated code.

Fixed navigation positioning

By default the positioning of main navigation is set to fixed. To change this behavior delete the fixed class from the main-navigation <div>:

<div class="main-navigation fixed">

Sliders

1. Main Slider

Main slider is implemented with Flexslider. Flexslider is an awesome, fully responsive jQuery slider toolkit.

You can find more documentation here http://www.woothemes.com/flexslider/

There are two variations for the main slider:

- Full Width Slider see index.html

- Boxed Width Slider see index-2.html

1.1 Full Slider

HTML markup:

<!-- slideshow start -->
<div class="slideshow fullwidth">

  <!-- flexslider start -->
  <div class="flexslider loading">
    <ul class="slides">

      <!-- first slide start -->
      <li style="background-image:url('images/slider-1-slide-1.png');">
        <div class="container">
          <div class="row">
            <div class="col-md-12">
              <div class="caption-wrapper">
                Caption content here
              </div>
            </div>
          </div>
        </div>
      </li>
      <!-- first slide end -->

      ........

    </ul>
   </div>
   <!-- flexslider end -->

</div>
<!-- slideshow end -->
1.2 Boxed Width Slider
HTML markup:
 
<!-- slideshow start -->
<div class="slideshow boxed-slideshow">

  <!-- flexslider start -->
  <div class="flexslider">
    <ul class="slides">

      <!-- first slide start -->
      <li>
        <a href=""><img src="" alt="slide 1"></a>
        <div class="caption">
          caption content here
        </div>
      </li>
      <!-- first slide end -->

      .......

    </ul>
  </div>
  <!-- flexslider end -->

</div>
<!-- slideshow end -->

3. Basic Content Slider

OWL Carousel is used to create simple content sliders or carousels see features-shortcodes.html.

Isotope Filtering

For the Filtering, iVega is using the Isotope plugin  http://isotope.metafizzy.co/

Here is an example:

Filters Options HTML markup:

<div class="filters">
  <ul class="nav nav-pills">
    <li class="active"><a href="#" data-filter="*">All</a></li>
    <li><a href="#" data-filter=".web-design">Web design</a></li>
    <li><a href="#" data-filter=".app-development">App development</a></li>
    <li><a href="#" data-filter=".site-building">Site building</a></li>
  </ul>
</div>

 

Filters Data HTML markup:

<div class="img-boxes one-third isotope-container">

  <div class="img-box isotope-item web-design">
    <a href="portfolio-item.html" class="mask-wrapper">
    <img src="images/portfolio-1.jpg" alt="">
    <div class="mask">
      <p>Lorem ipsum dolor sit amet</p>
      <span>Web Desing</span>
      <span class="triangle"><i class="fa fa-link"></i></span>
    </div>
    </a>
  </div>
  .......
</div>

Contact form

Regarding contact form as main part of Contact Us section, only thing you need to do is to setup your email address.
To do this, please open "php/contact-form.php" file and change this line:

$to = 'mail@domain.com';

One notice: this will be functional only on your server, php file can not be executed on your local computer.

Google Maps

iVega comes with a great Google maps integration. Making it easy for your visitor to locate your business on the map plays an ever-growing role nowadays.

To change the default Latitude, Longitude and Zoom of the Google map, please replace the following values to your js/template.js file:

myLatlng = new google.maps.LatLng(41.38791700, 2.16991870);
myZoom = 12;

Animations Effect

animate.css http://daneden.github.io/animate.css/

You can add some Animations / Transitions to the elements which fires when an element scrolls into view. Great for emphasis, home pages, sliders, and general just-add-water-awesomeness.

Usage:

<div class="object-non-visible" data-animation-effect="fadeInRight" data-effect-delay="200">
...
</div>

The Attribute data-effect-delay is Optional and is set in Miliseconds.

The animation can be set to:

  • bounce
  • flash
  • pulse
  • rubberBand
  • shake
